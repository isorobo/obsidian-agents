# Building Claude Code with Boris Cherny

> Gergely Orosz interviews Boris Cherny on the design and engineering of Claude
> Code. The Pragmatic Engineer Newsletter.

## Summary

Boris Cherny describes how the Claude Code team built an agentic coding tool. He
covers the design choices, the engineering culture, and the principles that
shaped the product.

## Key Concepts

- Small, sharp tools beat one large tool.
- The terminal is a durable surface for an agent.
- Fast iteration compounds when the team dogfoods the product.

## Terminology

- Agentic coding — an agent that reads, writes, and runs code under human review.

## Best Practices

- Keep the agent loop observable.
- Favour composable tools over monolithic modes.
