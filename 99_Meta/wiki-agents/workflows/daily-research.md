# Workflow — Daily Research

A short daily loop that keeps the vault current.

## Steps

1. Triage `00_Inbox/`. Convert each capture into a source or concept note, or
   discard it.
2. Run one or two watchlist channels. Start with `anthropic-news`.
3. Enrich two stubs. Promote `status: stub` notes toward `draft`.
4. Add cross-links. Every new note links to its topic MOC and three related
   notes.

## Time box

Twenty minutes. The loop favours consistency over volume.

## Verification

- New source notes carry `watchlist_channel`.
- No new orphan notes. Check the dashboard in `[[00_Home/index]]`.
